# Расширение Difan 6.0.x сервиса «Экспресс Платежи: ЕРИП»
Расширение CMS Difan для интеграции с сервисом «Экспресс Платежи». Расширение позволяет выставить счет в системе ЕРИП, получить и обработать уведомление о платеже в системе ЕРИП.
